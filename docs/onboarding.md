# Онбординг пользователей

## Роли в системе

| Роль         | Кто это               | Что может делать                                     |
| ------------ | --------------------- | ---------------------------------------------------- |
| **Клиент**   | Закупщик, байер       | Смотреть каталог, оформлять заказы                   |
| **Менеджер** | Менеджер по продажам  | Видеть заказы своих клиентов, настраивать оповещения |
| **Админ**    | Администратор системы | Полный доступ + управление пользователями            |

---

## Для администратора

### Создание пользователя

1. Зайти в **Пользователи** (в шапке сайта)
2. Нажать **Создать пользователя**
3. Заполнить:
   - **Имя** и **Email**
   - **Временный пароль** (минимум 8 символов) — пользователь сменит его при первом входе
   - **Роль**: Клиент / Менеджер / Администратор
   - **Менеджер** (только для Клиента) — к какому менеджеру прикреплён
   - **Группа цен** — прайс-лист, по которому клиент будет видеть цены
4. Нажать **Создать**

> При первом входе пользователь обязательно сменит пароль.

### Группы цен

Каждому клиенту назначается группа цен — это определяет, какой прайс-лист он видит:

| Группа              | Описание                   |
| ------------------- | -------------------------- |
| Прайс основной      | Базовый прайс              |
| Прайс 1 уровень     | Оптовый уровень 1          |
| Прайс 2 уровень     | Оптовый уровень 2          |
| Прайс Спот          | Спотовые цены              |
| Прайс Субы          | Субдистрибьюторы           |
| Прайс Градусы       | Сеть Градусы               |
| Прайс Пив.com       | Сеть Пив.com               |
| Прайс beer.exe      | Сеть beer.exe              |
| Прайс спец мьельнир | Специальный прайс Мьельнир |

Если группа цен не назначена — клиент видит "Прайс основной".

### Сброс пароля

1. В таблице пользователей нажать иконку 🔑 рядом с нужным пользователем
2. Ввести новый временный пароль
3. Нажать **Сбросить** — при следующем входе пользователь обязательно его сменит

### Деактивация пользователя

1. Нажать иконку редактирования ✏️
2. Переключить **Активен** в выключенное положение
3. Нажать **Сохранить** — пользователь больше не сможет войти

## Для менеджера

### Просмотр заказов клиентов

Раздел **Заказы** показывает все заказы клиентов, прикреплённых к тебе.

Статусы заказов:
| Статус | Значение |
|---------------|---------------------------------------|
| `Ожидает` | Новый заказ, ожидает обработки |
| `Подтверждён` | Принят в работу |
| `Отправлен` | Передан на доставку |
| `Доставлен` | Доставлен клиенту |
| `Отменён` | Отменён |

> Изменение статусов заказа производится вручную (или через интеграцию с МойСклад в следующей версии).

### Оповещения об остатках

Раздел **Оповещения** позволяет настроить уведомления когда остаток товара падает ниже порога.

1. Нажать **Добавить оповещение**
2. Выбрать товар и указать минимальный остаток (в штуках или кегах)
3. Нажать **Создать**

Система проверяет остатки каждый час. При нарушении порога на почту придёт письмо (не чаще 1 раза в 24 часа на оповещение).

Оповещение можно временно отключить переключателем без удаления.

## Для клиента

### Первый вход

1. Открыть сайт и войти с выданными логином и паролем
2. Система попросит сразу сменить пароль — придумай надёжный (минимум 8 символов)
3. Попадёшь в каталог товаров

### Работа с каталогом

- Товары сгруппированы по категориям — кликни на заголовок категории чтобы свернуть/развернуть
- Используй строку поиска вверху для быстрого поиска по названию
- Фильтр по категории — выбери нужную из списка слева

Что отображается в строке товара:

- **Название** — очищенное от технических пометок
- **Остаток** — количество (для кегов — целых кегов)
- **Срок годности** — дата окончания срока (если указана в МойСклад)
- **Цена** — по твоей группе цен
- **Объём** — для кегов (10, 20, 30 л.)

### Оформление заказа

1. Нажми **+** рядом с товаром или введи нужное количество в поле
2. Товар добавляется в корзину (иконка корзины в шапке)
3. Открой корзину, проверь список и количества
4. При необходимости добавь комментарий к заказу
5. Нажми **Оформить заказ**

Заказу присвоится номер вида `ORD-20260226-0001`. На почту менеджера придёт уведомление.

### Повтор заказа

В разделе **Заказы** → открой нужный заказ → нажми **Повторить заказ**. Система создаст новый заказ с теми же позициями.

### Отмена заказа

Отменить можно только заказы в статусе **Ожидает**. Открой заказ → **Отменить заказ**.

## Часто задаваемые вопросы

**Не вижу цены на товар**
— Скорее всего не назначена группа цен. Обратись к администратору.

**Товар есть в каталоге, но остаток 0**
— Товар закончился на складе. Данные обновляются каждые 15 минут.

**Забыл пароль**
— Обратись к администратору — он сбросит пароль и выдаст новый временный.

**Не приходят письма**
— Проверь папку "Спам". Если проблема сохраняется — обратись к администратору.
